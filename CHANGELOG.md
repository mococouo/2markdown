# Changelog

## 0.2.1

- Added a one-click `2Markdown.pyw` launcher for desktop users.
- Redesigned the desktop UI into a simpler user-facing flow with one large input area.
- Added dark mode, refreshed typography, softer visual styling, and page scrolling.
- Removed the visible log panel and simplified output selection.
- Kept English as the default language for new configurations.

## 0.2.0

- Added a polished Tkinter desktop app with drag-and-drop support when `tkinterdnd2` is installed.
- Added optional converter groups for Office, PDF, web, OCR, audio/video, email, markup, and GUI helpers.
- Added batch features for incremental runs, resume, dry run, archive recursion, glob filters, max file size, metadata, and frontmatter templates.
- Improved attachment extraction for Office, HTML, Markdown, EPUB, OpenDocument, notebooks, and PDFs where supported.
- Added multilingual UI support with English as the default language.

## 0.1.0

- Initial batch converter for common files to Obsidian-friendly Markdown.
- Added CLI, basic desktop launcher, conversion reports, frontmatter, and attachment output.
