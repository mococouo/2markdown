# Roadmap

## Near term

- Add screenshots and short demo GIFs for the desktop app.
- Add a packaged Windows release so non-technical users do not need Python.
- Improve PDF layout reconstruction for multi-column and slide-exported documents.
- Add a converter quality score in the report so users can find files that need review.

## Later

- Plugin template for third-party converters.
- More Obsidian-specific options, such as vault-relative attachment folders and tag presets.
- Optional local model support for summarizing imported notes.
- Signed Windows installer.
